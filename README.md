Este agente fue creado con la finalidad de hacer Scraping en la pagina del Boletin Oficial de la República Argentina. Su objetivo es el buscar todas las noticias novedodas de cierta fecha y luego ejecutar un análisis vía IA (en este caso con Ollama privilengiando el Open Sources) para determinar su importancia en empresas de tipo PSP o PSPCP.

Para ejecutar utilizar el comando: python bora_monitor.py

Para ejecutar en fecha especifica: python bora_monitor.py --fecha 00/00/0000

Para ejecutar sin utilizar análisis de IA: python bora_monitor.py --sin-ia

Saludos! :)
